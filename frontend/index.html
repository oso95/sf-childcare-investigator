Data sources

01 CCLD licensed facilities roster ArcGIS · direct
services.arcgis.com/XLPEppdz2H9dOiqp/arcgis/rest/services/CDSS_CCL_Facilities
California Community Care Licensing — authoritative for facility name, licensee, address, licensed capacity, program type, status. Scoped to SF county, CHILD CARE program.
02 SF Assessor parcel roll Hermai.ai
data.sfgov.org/resource/wv5m-vpq2.json
Secured property tax roll (closed-roll 2016). `property_area` is building sqft — the ground truth for Title 22 math. Resolved via canonical permits → block/lot join.
03 DBI building permits Hermai.ai
data.sfgov.org/resource/i98e-djp9.json
Every permit filed at the address. Reveals use changes (`existing_use` → `proposed_use`) and also serves as the address → block/lot lookup.
04 Registered business locations Hermai.ai
data.sfgov.org/resource/g8m3-pdis.json
SF Treasurer registry. Confirms the licensee is actually registered and surfaces NAICS category, dates, ownership.
05 DBI housing inspections Hermai.ai
data.sfgov.org/resource/nbtm-fbw5.json
Housing code complaints and Notices of Violation. Active NOVs at a licensed-facility address are a direct safety signal.
06 311 service requests Hermai.ai
data.sfgov.org/resource/vw6y-z8j6.json
Neighborhood complaints at the address — noise, illegal use, obstruction, building-without-permit.
07 Eviction notices Hermai.ai
data.sfgov.org/resource/5cei-gny5.json
Filed eviction notices per address — tenant-churn context when a facility expands into a mixed-use building.
Schemas pulled from data.sfgov.org via the Hermai.ai registry. Query recipes documented in the schema package.
